# Wildguide Description

This package contains the robot description files for the Wildguide project. It includes URDF files, meshes, and other resources needed for visualizing and simulating the robot in ROS2. The package is essential for defining the robot's physical properties and appearance, which can be used in RViz and Gazebo for visualization and simulation purposes.


## List of sensors and actuators
- GPS: For outdoor localization and navigation.
- IMU: For measuring the robot's orientation and acceleration.
- LIDAR: For obstacle detection and mapping.
- Camera: For visual perception and object recognition.
- Motor Controllers: For controlling the robot's movement based on navigation commands.
- Battery: For powering the robot's systems and components.
- Communication Module: For wireless communication and data transmission between the robot and the user interface.

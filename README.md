# wild-guide
Autonomous Navigation System for outdoor 

![Wild Guide](docs/media/wild_guide.png)

This repo is developed for Outdoor Navigation.

|ROS2 Packages|Description|
|---|---|
|wildguide_localization|ROS2 package for localization using GPS and IMU data.|
|wildguide_navigation|ROS2 package for navigation and path planning.|
|wildguide_control|ROS2 package for controlling the robot's movement based on navigation commands.|
|wildguide_perception|ROS2 package for processing sensor data and obstacle detection.|
|wildguide_interface|ROS2 package for user interface and interaction with the system.|
|wildguide_description|ROS2 package for robot description and visualization.|
|wildguide_simulation|ROS2 package for simulating the robot in a virtual environment.|
